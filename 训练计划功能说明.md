# 眼部训练应用 - 训练计划功能说明

## 🎉 新功能概览

训练计划管理系统已成功添加到眼部训练应用中！现在您可以创建、管理多个自定义训练计划，针对不同场景（工作间隙、深度训练、睡前放松）进行个性化训练。

## ✨ 核心功能

### 1. 训练计划管理
- **创建计划**：点击"➕ 新建计划"按钮创建新的训练计划
- **编辑计划**：点击计划卡片上的"✏️"按钮编辑现有计划
- **删除计划**：点击"🗑️"按钮删除不需要的计划
- **复制计划**：点击"📋"按钮快速复制现有计划并修改
- **导出计划**：点击"📤"按钮将计划导出为JSON文件
- **导入计划**：点击顶部"📥"按钮导入他人分享的计划

### 2. 模式配置
每个计划可以配置：
- **选择模式**：从8种训练模式中选择（水平、垂直、对角线、8字、随机、反弹等）
- **循环次数**：设置每个模式重复执行1-10次
- **持续时间**：配置每次执行5-120秒
- **速度调节**：为8字和反弹模式设置速度倍率
- **跳转间隔**：为随机模式设置位置跳转间隔

### 3. 训练执行
- **选择计划**：在"当前训练计划"下拉框中选择要使用的计划
- **开始训练**：点击"▶️ 开始训练"按钮，系统自动按计划顺序执行
- **进度显示**：训练时实时显示：
  - 📋 计划名称
  - 🎯 当前模式名称和图标
  - 循环进度（第X/Y次）
  - 整体任务进度（任务X/Y）

## 📦 预设计划

系统默认提供3个经典训练计划：

### 1. 全面训练
- **适用场景**：完整的眼部训练
- **包含模式**：全部8种模式
- **配置**：每个模式循环2次，每次15秒
- **总时长**：约4分钟

### 2. 快速5分钟
- **适用场景**：工作间隙快速缓解疲劳
- **包含模式**：水平、垂直、水平8字
- **配置**：每个模式1次，每次60秒
- **总时长**：3分钟

### 3. 缓解疲劳
- **适用场景**：睡前放松或深度缓解
- **包含模式**：水平8字、垂直8字、随机位置
- **配置**：8字模式速度0.8x，随机模式间隔4秒
- **总时长**：约2分钟

## 🚀 使用流程

### 创建自定义计划

1. 点击设置按钮（⚙️）打开设置面板
2. 滚动到"📋 训练计划"区域
3. 点击"➕ 新建计划"按钮
4. 填写计划名称和描述
5. 点击"➕ 添加模式"按钮
6. 在弹出的对话框中输入序号选择模式
7. 调整模式的循环次数、持续时间等参数
8. 使用"↑↓"按钮调整模式顺序
9. 点击"💾 保存计划"完成创建

### 使用训练计划

1. 在设置面板中找到"当前训练计划"下拉框
2. 选择您要使用的计划（或选择"普通模式"不使用计划）
3. 点击"▶️ 开始训练"按钮
4. 系统会按照计划配置自动执行训练
5. 训练过程中可以看到：
   - 当前计划名称
   - 正在执行的模式
   - 循环进度和整体进度
6. 完成所有训练后，系统会显示"🎉 训练完成！"

### 分享训练计划

1. 在计划卡片上点击"📤"导出按钮
2. 选择保存位置，下载JSON文件
3. 将文件分享给朋友
4. 朋友点击"📥"导入按钮
5. 选择JSON文件即可导入计划

## 💡 使用技巧

1. **场景化计划**：为不同场景创建专用计划
   - 上午工作前：全面训练计划
   - 午休时间：快速5分钟计划
   - 睡前放松：缓解疲劳计划

2. **渐进式训练**：创建不同强度的计划
   - 初级：较少循环次数，较长单次时间
   - 中级：适中配置
   - 高级：更多循环次数，更快速度

3. **计划复制**：基于现有计划快速创建变体
   - 点击"📋"复制按钮
   - 修改部分参数即可

4. **灵活切换**：随时在普通模式和计划模式之间切换
   - 选择"普通模式（不使用计划）"回到传统训练方式

## 🔧 技术细节

### 数据存储
- 所有计划数据保存在浏览器的LocalStorage中
- 键名：`eyeTrainingPlans`
- 数据格式：JSON
- 自动持久化，刷新页面不丢失

### 计划结构
```json
{
  "id": "plan-xxx",
  "name": "计划名称",
  "description": "计划描述",
  "modes": [
    {
      "type": "horizontal",
      "loops": 2,
      "duration": 15,
      "speed": 1.0
    }
  ]
}
```

### 执行队列
- 系统会将计划展开为线性执行队列
- 每个模式的循环次数会被展开为多个任务
- 按顺序逐个执行，完成后切换到下一个

## 📝 注意事项

1. **循环次数限制**：单个模式最多循环10次
2. **模式数量**：建议单个计划不超过20个模式配置
3. **参数验证**：系统会自动验证并修正超出范围的参数
4. **计划名称**：最长50个字符
5. **计划描述**：最长200个字符
6. **数据备份**：建议定期导出重要计划作为备份
